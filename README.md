# Breast Cancer Detection using Logistic Regression


## Project Description

Machine Learning project to classify breast tumors as:

- Malignant (M)
- Benign (B)


## Dataset Features

The dataset contains medical measurements:

- radius_mean
- texture_mean
- perimeter_mean
- area_mean
- smoothness_mean
- compactness_mean


## Algorithm Used

Logistic Regression


## Steps

1. Data Loading
2. Data Cleaning
3. Feature Scaling
4. Train/Test Split
5. Model Training
6. Evaluation

- Pandas

The model was evaluated using:
